


A fork of wkx-sharp by Cristian Schwarz.

This fork adds support for Spatialite Geometries

Currently builds with VS 2017
No tests have been added for Spatialite

A WKT/WKB/EWKT/EWKB/Spatialite parser and serializer with support for

- Point
- LineString
- Polygon
- MultiPoint
- MultiLineString
- MultiPolygon
- GeometryCollection
- CircularString
- CompoundCurve
- CurvePolygon
- MultiCurve
- MultiSurface
- PolyhedralSurface
- TIN
- Triangle
